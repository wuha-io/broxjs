
# API : contextMenus

## Manifest

To use this API you need to have the "contextMenus" permission.

```JSON
{
  "name": "My extension",
  "permissions": [
    "contextMenus" // require
  ],
  "icons": {
    "16": "icon-bitty.png", // require
    "48": "icon-small.png",
    "128": "icon-large.png"
  }
}
```
